# 👗 fAIshion – AI 기반 체형 맞춤 가상 피팅 & 쇼핑 플랫폼

> “입어보는 듯한 쇼핑 경험을 온라인에서”  
AI 스타일링, 가상 피팅, 결제 시스템, 재고·배송 관리까지 통합한 풀스택 패션 커머스 플랫폼

---

## 📌 프로젝트 개요

온라인 쇼핑이 늘어나면서 **사이즈 불일치로 인한 반품 증가**, **의류 폐기물 증가**,  
그리고 **쇼핑 경험의 불편함**이 대표적인 문제로 대두되고 있습니다.

fAIshion은 이러한 문제를 해결하기 위해 탄생한 **AI 기반 가상 피팅 플랫폼**으로,  
사용자의 체형·취향·날씨 데이터를 활용해 **맞춤형 상품 추천 + 가상 착용 경험**을 제공합니다.

---

## 🎯 주요 목표
- AI 기반 가상 피팅으로 **반품률 감소**
- 개인 맞춤형 상품 추천으로 **구매 전환율 상승**
- 불필요한 구매 감소로 **지속 가능한 패션 소비 환경 조성**
- 판매자/관리자 기능 통합으로 **운영 효율 극대화**

---

## 🙋 사용자 시나리오

### ▶ 구매자
- 회원가입 / 소셜 로그인  
- 상품 추천 배너에서 **AI 기반 스타일링 미리보기 제공**  
- 장바구니 → 결제 → 배송 추적

### ▶ 판매자
- 상품 등록 및 재고 관리  
- 상품 할인 설정, 판매종료 처리  
- 신고 리뷰 확인 및 상태 조치

## 🧱 기술 스택

### **Backend**
- Spring Boot  
- Spring Security + JWT  
- JPA, MySQL  
- 비관적 락(Pessimistic Lock) 기반 재고 관리  
- 상태조회 API, 사업자 등록 API

### **Frontend**
- React  
- REST API 통신  
- Styled Components / Custom UI

### **AI & API**
- Google Gemini 이미지 생성 API  
- Toss Payments API  
- 네이버 소셜 로그인  
- 우편번호 API

---

## 🧩 주요 기능

### 🔐 **Spring Security + JWT 인증 구조**
- Access / Refresh Token 분리  
- HttpOnly 쿠키 기반 보안 처리  
- Filter Chain 기반 권한 통제

### 🎨 **AI 스타일링 & 가상 피팅**
- 사용자 체형 기반 이미지 생성  
- 상품 추천 배너 자동 생성  
- 상세페이지 → AI 스타일링 즉시 연동

### 💳 **결제 기능**
- 신용/체크카드, 간편결제(카카오페이/네이버페이) 등 다중 결제수단  
- 주문–결제–정산 자동화 구조

### 📦 **재고 관리 + 배송 시스템**
- 비관적 락 적용 → 데이터 정합성 보장  
- 판매자 배송 처리·송장 입력  
- 구매자는 배송 상태 실시간 확인

### ⭐ **리뷰·문의·신고 관리**
- 사용자 신고 접수 → 관리자 조치  
- 마이페이지에서 본인 작성 리뷰 확인 가능
